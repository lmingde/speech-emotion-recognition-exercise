# speech-emotion-recognition-exercise
2018年7⽉30⽇-8⽉13⽇持续2周的AI训练营中语⾳情感识别营的项目报告。

* 项目说明：
本次项目提供教学场景下的10秒音频片段，每个音频片段对应一个P值，P值-3到3之间，P值越大表示愉悦度程度越高。本项目通过音频的声学特征，对P值进行预测。（回归任务）
* 第一周：机器学习方法
* 第二周：深度学习做法
